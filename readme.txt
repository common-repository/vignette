This plugin adds a vignette to your website.

== Description ==

A vignette is an effect used in photography. It basically darkens the edges of a photo and is sometimes undesired but in a lot of cases it's being "added" by using a filter or afterwards using software.

== Installation ==

1.	Install the directory in the wp-content/plugins folder
2.	Activate the plugin
3.	Change the settings (if needed)

You're all done!
